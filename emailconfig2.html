configuration of the Mail Server

The main configuration for Postfix Linux mail server in the /etc/postfix/main.cf file.

The files include various options which include;

  • myhostname = mail.mydomain.com- to specify the hostname of the mail server, which is where postfix will obtain its emails.
  • myorigin = $exampledomain.com- All emails sent from this mail server will look as though they came from the one that you specify in this option.
  • mydestination = $myhostname, localhost.$exampledomain.com- shows you which domains the Postfix server uses for incoming emails to your Linux email server.

$ When you change any of files used to configure Linux mail server for Postfix, you must reload the service, with this directive:

$ systemctl reload postfix

Distraction and typing things in can often result in mistakes, but you can track down any misspellings that might compromise your Linux mail server using this command:

$ postfix check
